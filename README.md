# Solo Leveling English

**Solo Leveling English** is a premium English academy landing page designed for the Korean education market. It focuses on School Exams (Naesin) and College Entrance Exams (Suneung) with a "Level Up" concept.

## Features

- **Localized Content**: Fully translated into Korean.
- **Responsive Design**: Optimized for Desktop (sticky header, hero, grid services) and Mobile.
- **Modern UI**: Dark/Navy/Gold color scheme with 3D illustrations.
- **External Integrations**:
  - **Consultation**: Links to KakaoTalk Open Chat.
  - **Locations**: Links to Naver Map.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Font**: Geist Sans / Geist Mono

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

- `src/app`: App directory (Layout, Page, Globals.css)
- `src/components/landing`: Landing page components (Header, Hero, Services, Footer)
- `public`: Static assets (Hero images)
